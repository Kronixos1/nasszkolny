Kwasy karboksylowe:
kwas metanowy = mrówkowy HCOOH
kwas etylowy = octowy CH3COOH
kwas propanowy = propionowy C2H5COOH
kwas butanowy = masłowy C3H7COOH
kwas pentanowy = walerianowy C4H9COOH

kwas Xnowy = C[x-1]-H[1+x]-COOH 


Alkohole: 
metanol CH3-OH = CH3OH
etanol CH3-CH2-OH = C2H5OH
propanol CH3-CH2-CH2-OH = C3H7OH
butanol	CH3-CH2-CH2-CH2-OH = C4H9OH

Xnol = CH3-[CH2*(X-1)]-OH
Xnol = C[x]H[2x+1]OH

ESTRY:
Kwas + Alkohol ----> Ester

HCOOH + CH3OH => CH3COOH + H2O

kw. mrówkowy + metanol => mrówczan metanolu + woda
^===v
kw. metanowy + metanol => metanian metylu + woda

kw. karboksylowy + alkohol => [nazwa kwasu + an/ian] + dopełniacz alkholu + H2O



Estry nadają zapach perfumą, kwiatą itd.

Proces estryfikacji to proces otrzymywania estrów danej substancji
Przykłady:

octan butylu
CH3COOH + C4H90H => CH3COOC4H9 + H2O

pentanian propylu
C4H9COOH + C3H7OH => C4H9COOC3H7 + H2O

maślan hexanu
C3H7COOH + C6H13OH => C3H7COOC6H13 + H20

Xnowy + Xnol => [Xnowy(-H) + Xnol(-OH) + H2O]


...... =O
R1 - C
......- O - R2

R1 = grupa węglowodrowa pochodząca od kw. karboksylowego
R2 = grupa węglowodorwa pochodząca od alkoholu

Krótkie są rozpuszczalne w wodzie a długie nie
